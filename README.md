## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

### 取得專案

```bash
git clone https://github.com/jaosn60810/meme-image-vuetify-project.git
```

### 移動到專案內

```bash
cd meme-image-vuetify-project
```

### 安裝套件

```bash
pnpm install
```

### 運行專案

```bash
pnpm dev
```

### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://127.0.0.1:3000/
```

## 資料夾說明

- views - 畫面放置處
- router - 路由放置處
- components - 元件放置處
- store - 狀態放置處
- assets - 靜態資源放置處

## 專案技術

- Vue v3.2.13
- Vite v3.1.9
- Vue Router v4.0.0
- Vuetify v3.0.0
